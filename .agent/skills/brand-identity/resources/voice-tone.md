# Copywriting: Voice & Tone Guidelines

**Brand:** EmpathoAI
**Core Identity:** Outcome as a Service (OaaS) Strategic Orchestrator

When generating text, adhere to this brand persona.

## Brand Personality Keywords
* **Intellectual & Sovereign:** High-level strategic depth. We speak as a board of directors.
* **Brutal Clarity:** Direct, efficient, and results-focused. No unnecessary adjectives.
* **Institutional Rigor:** Cut through market "noise" with absolute technical and strategic rigor.
* **Professional but approachable** (Sovereign yet accessible).

## Grammar & Mechanics Rules
* **Directness:** Use active voice. Be authoritative.
* **Headings:** Use Title Case for main headings (H1, H2). Use sentence case for subheadings (H3+).
* **Punctuation:** Avoid exclamation points (!) in institutional copy. Use periods for complete sentences.
* **Clarity:** Keep sentences concise.

## Terminology Guide

| Do Not Use | Use Instead |
| :--- | :--- |
| "Utilize" | "Use" |
| "In order to..." | "To..." |
| "Medical/Clinic" | "Sovereign/Outcome" |
| "Concierge" | "Orchestrator" |
| "Market Noise" | "Institutional Rigor" |

## Visual Language
References to **"Void"** (the starting state) and **"Rigor"** (the path to outcomes).
